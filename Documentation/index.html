A. About the Project

Crud PHP Mini Project Project. Designed & Developed this project using HTML5, CSS, PHP & MySQL. Also, very user-friendly & cross browser compatibility. Using custom Pagination, Trash & Retore mechanism. Also, using text field, radio button, drop-down list & checkbox.

B. Installation

First of all, download any local server like wamp or xampp from below links.

Xampp Download Link : Download

Wamp Download Link : Download

After completed downloading install xampp or wampp server any drive on your computer. Go to the root directory on your server. If you are installing xampp go to the htdocs directory, for wamp server go to the www directory.

Put here project folder.

Here is sample screenshot for xampp server.

Then go to the any of your favorite web browser and write localhost/phpmyadmin

From New create a database.

From the phpmyadmin click on import tab importing your .sql file from your project folder.

C. Configuration

Now go to your project folder. From config directory edit config.php

After configure this go to web browser and write localhost/your_project_directory_name Example: localhost/comment_system

Have Fun !

D. Examples

Some of project examples are given below:-

index page : This this project main page. Index page is just welcome message show.

create page : This this create page. From this page you can create data.

display page : This this display page. From this page you can show all created data and also manage edit, delete data. If you are deleting data it will go to trash page.

trashlist page : This this trashlist page. From this page you can show all trashed data and also manage restore, permenantly deleted data. If you are deleting data it will permenantly deleted.
